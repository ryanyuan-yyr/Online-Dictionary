test run
tɛst rʌn
Word forms: plural test runs
N-COUNT (机器、系统的)试运行，初步试验 If you give a machine or system a test run, you try it out to see if it will work properly when it is actually in use. (机器、系统的)试运行，初步试验

  • Japan's space ambitions have had a set-back after the failure of an engine in a test run.

    一次试运行中的引擎失灵使得日本的太空梦想遭受了挫折。

  • The vote was seen as a test run for elections to come.

    这次投票被看作是即将到来的选举的预演。
