interface
ɪntəʳfeɪs ˈɪntərˌfeɪs
Word forms: plural , 3rd person singular present tense interfaces, present participle interfacing, past tense , past participle interfaced
1 N-COUNT (学科或系统间相互影响或衔接的)边缘区域,接合部位 The interface between two subjects or systems is the area in which they affect each other or have links with each other. (学科或系统间相互影响或衔接的)边缘区域,接合部位

  • ...a witty exploration of that interface between bureaucracy and the working world. [+ between]

    对官僚阶层与劳动大众之间的临界区域的风趣探究

SYN connection, link, boundary, border  
2 N-COUNT [usually n NOUN] (软件的用户)界面 If you refer to the user interface of a particular piece of computing software, you are talking about its presentation on screen and how easy it is to operate. (软件的用户)界面 [computing]

  • ...the development of better user interfaces.

    更友好的用户界面的开发

3 N-COUNT (计算机或电子设备的)接口 In computing and electronics, an interface is an electrical cricuit which links one machine, especially a computer, with another. (计算机或电子设备的)接口 [technical]
4 V-RECIP-ERG (使)(与…)相互联系（或连接） If one thing interfaces with another, or if two things interface, they have connections with each other. If you interface one thing with another, you connect the two things. (使)(与…)相互联系（或连接） [technical, formal] [Also VERB n]

  • ...the way we interface with the environment. [VERB + with]

    我们与环境相互作用的方式

  • The different components all have to interface smoothly. [VERB]

    不同的部件都必须能顺利接合。

  • He had interfaced all this machinery with a master computer. [VERB noun + with]

    他把所有机器连接到了一台主计算机上。

SYN connect, couple, link, combine
