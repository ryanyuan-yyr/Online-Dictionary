run by
PHRASAL VERB (为征求意见）把（某事）告诉（某人） If you run something by someone, you tell them about it or mention it, to see if they think it is a good idea, or can understand it. (为征求意见）把（某事）告诉（某人）

  • I'm definitely interested, but I'll have to run it by Larry Estes. [VERB noun PREP. noun]

    我很有兴趣，但是我还得告诉拉里·埃斯蒂斯一声。

  • Run that by me again. [VERB noun PREP. noun]

    再把那件事跟我说一遍。
