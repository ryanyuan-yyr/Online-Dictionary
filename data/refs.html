ref
ref rɛf
Word forms: plural refs
1 档案编号;文件编号 Ref is an abbreviation for reference. It is written in front of a code at the top of business letters and documents. The code refers to a file where all the letters and documents about the same matter are kept. 档案编号;文件编号

  • Our Ref: JAH/JW.

    我们的编号: JAH/JW。

2 N-COUNT （比赛中的）裁判 The ref in a sports game, such as football or boxing, is the same as the referee. （比赛中的）裁判 [informal]

  • The ref gave a penalty and Zidane scored.

    裁判判罚点球，齐达内操刀命中。
