type in
(also type into)
PHRASAL VERB 把(信息)输入（计算机） If you type information into a computer or type it in, you press keys on the keyboard so that the computer stores or processes the information. 把(信息)输入（计算机）

  • Officials type each passport number into a computer. [VERB noun PREP. noun]

    行政人员将每个护照号都输入计算机中。

  • You have to type in commands, such as 'help' and 'print'. [VERB PREP. noun]

    你必须输入命令，例如“帮助”或“打印”。

  • You type things in, and it responds. [VERB noun PREP.]

    你把东西输入进去，它就会作出反应。
