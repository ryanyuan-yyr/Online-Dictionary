query
kwɪəri ˈkwɪri ˈkwɛri
Word forms: plural , 3rd person singular present tense queries, present participle querying, past tense , past participle queried
1 N-COUNT （尤指向组织、出版物或专家提出的）问题，疑问，询问 A query is a question, especially one that you ask an organization, publication, or expert. （尤指向组织、出版物或专家提出的）问题，疑问，询问

  • If you have any queries about this insurance, please contact Travel Insurance Services Limited. [+ about]

    如果您对此项保险有什么疑问，请与旅行保险服务有限公司联系。

SYN question, inquiry, enquiry, problem  
2 VERB 对…提出疑问;对…表示怀疑 If you query something, you check it by asking about it because you are not sure if it is correct. 对…提出疑问;对…表示怀疑

  • It's got a number you can ring to query your bill. [VERB noun]

    这上面有一个号码，您可以打电话查询您的账单。

  • No one queried my decision. [VERB noun]

    没人对我的决定提出质疑。

SYN question, challenge, doubt, suspect  
3 VERB 提问;发问;询问 To query means to ask a question. 提问;发问;询问 [Also VERB n]

  • 'Is there something else?' Ryle queried as Helen stopped speaking. [VERB with quote]

    海伦的话音刚落，赖尔便问道：“还有其他事吗？”

  • One of the journalists queried whether sabotage could have been involved. [VERB wh]

    其中一位新闻记者询问是否有人为破坏的可能性。

SYN ask, inquire or enquire, question
