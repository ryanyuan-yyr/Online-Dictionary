existing
ɪgzɪstɪŋ
ADJ [ADJ n] （尤指与未来相对的）现存的，现行的，目前的 Existing is used to describe something which is now present, available, or in operation, especially when you are contrasting it with something which is planned for the future. （尤指与未来相对的）现存的，现行的，目前的

  • ...the need to improve existing products and develop new lines.

    改进现有产品、开发新生产线的需要

  • Existing timbers are replaced or renewed.

    现有的木料进行了替换或翻新。

  • This facility is open to both new and existing borrowers.

    这套设备向新老借用者开放。

SYN in existence, living, present, surviving
