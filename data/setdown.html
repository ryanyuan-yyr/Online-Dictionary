set down
1 PHRASAL VERB 制定，规定(规则等) If a committee or organization sets down rules for doing something, it decides what they should be and officially records them. 制定，规定(规则等) [Also VERB n PREPOSITION]

  • The Convention set down rules for deciding which country should deal with an asylum request. [VERB PREP. noun]

    会议制定了用以确定由哪个国家负责受理避难请求的规约。

2 PHRASAL VERB 把(思想、经历等)写下来;记下 If you set down your thoughts or experiences, you write them all down. 把(思想、经历等)写下来;记下 [Also VERB n PREPOSITION]

  • Old Walter is setting down his memories of village life. [VERB PREP. noun]

    老沃尔特正在写他乡村生活的回忆录。

SYN write down, record, jot down, draft
