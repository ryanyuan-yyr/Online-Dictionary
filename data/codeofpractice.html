code of practice kəʊd əv ˈpræktɪs
Word forms: plural codes of practice
N-COUNT 行业规则；从业准则 A code of practice is a set of written rules which explains how people working in a particular profession should behave. 行业规则；从业准则

  • The auctioneers are violating a code of practice by dealing in stolen goods.

    拍卖商买卖赃物，违反了行规。
