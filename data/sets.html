Word forms: plural sets

NOUN USES 名词用法

1 N-COUNT 一套;一组;一系列 A set of things is a number of things that belong together or that are thought of as a group. 一套;一组;一系列

  • There must be one set of laws for the whole of the country. [+ of]

    整个国家必须要有一套统一的法律。

  • I might need a spare set of clothes. [+ of]

    我可能需要一套备用的衣服。

  • The computer repeats a set of calculations. [+ of]

    计算机重复着一组运算。

  • Only she and Mr Cohen had complete sets of keys to the shop. [+ of]

    只有她和科恩先生有店里的全套钥匙。

  • The mattress and base are normally bought as a set.

    通常床垫和床架是成套购买的。

  • ...a chess set.

    一副国际象棋

SYN series, collection, assortment, kit  
2 N-COUNT (网球比赛中的)盘 In tennis, a set is one of the groups of six or more games that form part of a match. (网球比赛中的)盘

  • Graf was leading 5-1 in the first set.

    格拉芙第一盘以 5 比 1 领先。

3 N-COUNT (数学中的)集，集合 In mathematics, a set is a group of mathematical quantities that have some characteristic in common. (数学中的)集，集合
4 N-COUNT (一场音乐会中所演唱或演奏的)一组歌曲(或乐曲) A band's or musician's set is the group of songs or tunes that they perform at a concert. (一场音乐会中所演唱或演奏的)一组歌曲(或乐曲)

  • The band continued with their set after a short break.

    短暂的休息后，乐队继续演奏他们的曲目。

  • He plays a solo acoustic set.

    他伴着原声乐器独唱了一组歌曲。

5 N-SING (经常碰面或趣味相同的)一伙(或一帮、一群)人 You can refer to a group of people as a set if they meet together socially or have the same interests and lifestyle. (经常碰面或趣味相同的)一伙(或一帮、一群)人

  • He belonged to what the press called 'The Chelsea Set'.

    他属于媒体所称的“切尔西圈子”里的人。

6See also:jet set 
7 N-COUNT [oft on/off N] (话剧的)布景;(电影的)摄影棚，拍片现场 The set for a play, film, or television show is the furniture and scenery that is on the stage when the play is being performed or in the studio where filming takes place. (话剧的)布景;(电影的)摄影棚，拍片现场

  • From the first moment he got on the set, he wanted to be a director too.

    自从进入摄影棚的第一刻起，他就希望自己也能成为一名导演。

  • ...his stage sets for the Folies Bergeres. [+ for]

    他为牧女游乐园剧院设计的舞台布景

  • ...a movie set.

    电影拍摄现场

  • ...stars who behave badly on set.

    在拍摄现场表现糟糕的明星们

SYN scenery, backdrop, setting, flats  
8 N-SING (尤指坚定的)神情，姿态，姿势 The set of someone's face or part of their body is the way that it is fixed in a particular expression or position, especially one that shows determination. (尤指坚定的)神情，姿态，姿势

  • Matt looked at Hugh and saw the stubbornness in the set of his shoulders. [+ of]

    马特看着休，从他坚挺的肩膀中看出了他的倔强。

  • Artist Richard Stone has captured in her eyes and the set of her face her steely determination.

    画家理查德·斯通在她的眼神和面部表情中捕捉到了她的坚毅和决绝。

SYN expression, look  
9 N-COUNT (电视机等)家用电器 A set is an appliance. For example, a television set is a television. (电视机等)家用电器

  • Children spend so much time in front of the television set.

    孩子们花在看电视上的时间太多了。

  • We got our first set — black and white — in 1963.

    1963 年我们买了我们的第一台电视机—黑白的。

SYN television, TV, telly [British , informal] , the box [British , informal]  

VERB AND ADJECTIVE USES 动词和形容词用法

The form set is used in the present tense and is the past tense and past participle of the verb. set 的过去式和过去分词与原形相同。
1 VERB (尤指小心翼翼地)放，置 If you set something somewhere, you put it there, especially in a careful or deliberate way. (尤指小心翼翼地)放，置

  • He took the case out of her hand and set it on the floor. [VERB noun prep.]

    他从她手中拿过箱子，将它小心地放在地板上。

  • When he set his glass down he spilled a little drink. [VERB noun with adv.]

    他把杯子放下时，饮料溅出来一点。

SYN put, place, lay, leave  
2 ADJ 位于…的;坐落在…的 If something is set in a particular place or position, it is in that place or position. 位于…的;坐落在…的

  • The castle is set in 25 acres of beautiful grounds. [+ in]

    这座城堡坐落于一片方圆 25 英亩、风景秀丽的土地上。

  • Quiberon is set on an eight-mile peninsula.

    基伯龙位于一个 8 英里大小的半岛上。

SYN located, sited, situated, found  
3 ADJ 镶于…上的;嵌在…上的 If something is set into a surface, it is fixed there and does not stick out. 镶于…上的;嵌在…上的

  • The man unlocked a gate set in a high wall and let me through. [+ in]

    那名男子打开了高墙中一面大门上的锁让我进去。

  • Set into an alcove under the side deck is a tiny wash basin.

    舷边甲板下面的一个凹处放了一个小小的洗脸盆。

4 VERB 使处于(某种状态或情况) You can use set to say that a person or thing causes another person or thing to be in a particular condition or situation. For example, to set someone free means to cause them to be free, and to set something going means to cause it to start working. 使处于(某种状态或情况)

  • Set the kitchen timer going. [VERB noun verb-ing]

    让厨房计时器开始计时。

  • A phrase from the conference floor set my mind wandering. [VERB noun verb-ing]

    会上发言者的一句话让我思绪飘远。

  • Dozens of people have been injured and many vehicles set on fire. [be V-ed adj/adv]

    很多人受了伤，多辆车起火。

  • Churchill immediately set into motion a daring plan. [V n with prep]

    丘吉尔立刻开始实施一项大胆的计划。

5 VERB 设置，调整好，调准(钟表等) When you set a clock or control, you adjust it to a particular point or level. 设置，调整好，调准(钟表等)

  • Set the volume as high as possible. [VERB noun adv./prep.]

    将音量尽量调大。

  • I forgot to set my alarm and I overslept. [VERB noun]

    我忘了定闹钟,结果睡过了头。

SYN switch on, turn on, activate, programme  
6 VERB 决定，确定(日期、价格、目标或水准) If you set a date, price, goal, or level, you decide what it will be. 决定，确定(日期、价格、目标或水准)

  • The conference chairman has set a deadline of noon tomorrow. [VERB noun]

    会议主席将明天中午定为截止日期。

  • A date will be set for a future meeting. [be VERB-ed + for]

    将会确定将来会议的时间。

  • The German government has set a tight budget for next year. [VERB noun]

    德国政府已经为明年制定了紧缩的财政预算。

  • The pass mark is set at 50 per cent. [be V-ed at n]

    合格分数设定为百分制的 50分。

SYN arrange, decide (upon), settle, name  
7 VERB 估定(价值);认为有（…价值) If you set a certain value on something, you think it has that value. 估定(价值);认为有（…价值)

  • She sets a high value on autonomy. [VERB noun + on]

    她对自主权看得很重。

  • If you set no value on being a woman yourself, how can you expect others to? [V n on n/-ing]

    如果你自己都认为做女人一文不值，又怎么能期望别人尊重你呢？

8 VERB 树立(榜样);创造(纪录);开创(先例) If you set something such as a record, an example, or a precedent, you do something that people will want to copy or try to achieve. 树立(榜样);创造(纪录);开创(先例)

  • Legal experts said her case would not set a precedent because it was an out-of-court settlement. [VERB noun]

    法律专家说她的案子不会成为判例，因为它是庭外和解的。

  • A new world marathon record of 2 hrs, 8 min, 5 sec, was set by Stephen Jones of Great Britain. [be VERB-ed]

    英国的斯蒂芬·琼斯创造了新的马拉松世界纪录— 2 小时 8 分零 5 秒。

  • They set the pace in cutting ozone-damaging emissions. [VERB noun]

    他们在减少破坏臭氧层的气体排放量方面走在了最前面。

  • If you are smoking in front of the children then you are setting them a bad example. [VERB noun noun]

    如果你在孩子们面前抽烟，那你就给他们树立了一个坏榜样。

SYN create, provide, establish, set up  
9 VERB 分配;布置;指派 If someone sets you a task or aim or if you set yourself a task or aim, you need to succeed in doing it. 分配;布置;指派

  • I have to plan my academic work very rigidly and set myself clear objectives. [VERB noun noun]

    我必须严格地规划我的学业，同时给自己定下明确的目标。

  • We will train you first before we set you a task. [VERB noun noun]

    我们在给你分派任务之前会先对你进行培训。

  • The secret to happiness is to keep setting yourself new challenges. [VERB noun noun]

    幸福的秘诀就是要不停地给自己设定新的挑战。

SYN assign, give, allot, prescribe  
10 VERB 出(试题、试卷) To set an examination or a question paper means to decide what questions will be asked in it. 出(试题、试卷) [British]

  • He broke with the tradition of setting examinations in Latin. [VERB noun]

    他打破了用拉丁语出试卷的传统。

in AM, usually use 美国英语通常用 make up
11 ADJ [usually ADJ n] 安排好的;确定的;固定的 You use set to describe something which is fixed and cannot be changed. 安排好的;确定的;固定的

  • Investors can apply for a package of shares at a set price.

    投资者可以申购固定价格的股票组合。

  • A set period of fasting is supposed to bring us closer to godliness.

    据说定期斋戒会让我们更虔诚。

  • There is a set menu from £4.00 for two courses with coffee.

    有最低消费是4英镑的两道菜加咖啡的套餐。

SYN established, fixed, specified, planned  
12 ADJ [ADJ n] (学习书目)指定的 A set book must be studied by students taking a particular course. (学习书目)指定的 [British]

  • One of the set books is Jane Austen's Emma.

    指定阅读书目之一是简·奥斯汀的《爱玛》。

SYN compulsory, required, assigned, recommended  
in AM, use 美国英语用 required
13 ADJ (戏剧、电影、小说等)以(某时或某地)为背景的 If a play, film, or story is set in a particular place or period of time, the events in it take place in that place or period. (戏剧、电影、小说等)以(某时或某地)为背景的

  • The play is set in a small Midwestern town. [+ in]

    这出戏以中西部的一个小镇为背景。

  • ...a 1964 science fiction novel by Philip K Dick, set in 1994 in a colony of humans on Mars.

    菲利普·K.迪克 1964 年写的一部以 1994 年火星上一个人类聚居地为背景的科幻小说。

  • The Hungarian director has completed her powerful Diary trilogy, set against the background of events in her country.

    这名匈牙利导演已经完成了她震撼人心的“日记三部曲”，故事是以发生在她本国的事件为背景。

14 ADJ 做好准备的;可能的 If you are set to do something, you are ready to do it or are likely to do it. If something is set to happen, it is about to happen or likely to happen. 做好准备的;可能的

  • Roberto Baggio was set to become one of the greatest players of all time.

    罗伯特·巴乔注定会成为迄今为止最伟大的球员之一。

  • The talks are set to continue through the week. [VERB noun]

    谈判可能会持续一周。

15 ADJ 下定决心的;坚决的 If you are set on something, you are strongly determined to do or have it. If you are set against something, you are strongly determined not to do or have it. 下定决心的;坚决的

  • She was set on going to an all-girls school. [+ on/against]

    她执意要去女子学校上学。

  • Margaret was always mischievous and set on her own individual course.

    玛格丽特总是调皮捣乱，而且一意孤行。

  • France is also set against devaluation.

    法国也坚决反对货币贬值。

16 VERB 使表现出，显出(坚定的表情) If you set your face or jaw, you put on a fixed expression of determination. 使表现出，显出(坚定的表情)

  • Instead, she set her jaw grimly and waited in silence. [VERB noun]

    相反，她一脸严肃，面部紧绷，一声不吭地等着。

  • He came insolently towards Mr. Won, his features set in a scowl. [VERB noun]

    他傲慢无礼地朝元先生冲过来，面露怒容。

17 VERB 凝固;凝结;变硬 When something such as jelly, melted plastic, or cement sets, it becomes firm or hard. 凝固;凝结;变硬

  • You can add ingredients to these desserts as they begin to set. [VERB]

    这些甜点开始凝固时就可以加入辅料了。

  • Lower the heat and allow the omelet to set on the bottom. [VERB]

    关小火让煎蛋的下面凝固。

  • The material requires higher temperatures and pressures to set hard. [VERB adj.ive]

    这种材料需要有更高的温度和压力才能凝结变硬。

SYN harden, stiffen, condense, solidify  
18 VERB (太阳)落山，落下 When the sun sets, it goes below the horizon. (太阳)落山，落下

  • They watched the sun set behind the distant dales. [VERB]

    他们望着夕阳没入远处的山谷。

  • ...the red glow of the setting sun. [VERB-ing]

    落日的红色余晖

19 VERB 设置，安排，设计(陷阱) To set a trap means to prepare it to catch someone or something. 设置，安排，设计(陷阱)

  • He seemed to think I was setting some sort of trap for him. [VERB noun + for]

    他似乎认为我在给他设什么套儿。

  • They dug trenches in their path and set booby traps. [VERB noun]

    他们在小路上挖深沟设饵雷。

20 VERB 在(桌)上摆放餐具 When someone sets the table, they prepare it for a meal by putting plates and cutlery on it. 在(桌)上摆放餐具
21 VERB 为…谱(曲);为…配(乐) If someone sets a poem or a piece of writing to music, they write music for the words to be sung to. 为…谱(曲);为…配(乐)

  • He has attracted much interest by setting ancient religious texts to music. [VERB noun + to]

    他因为给古老的宗教经书谱曲配乐而备受关注。

22 See also: setting  set-to 
23 PHRASE 为…做准备;为…铺平了道路 If someone sets the scene or sets the stage for an event to take place, they make preparations so that it can take place. 为…做准备;为…铺平了道路

  • The Democrat convention has set the scene for a ferocious election campaign this autumn.

    民主党代表大会已经为今年秋季激烈的大选作好了准备。

  • The company has been setting the stage recently for progress in the US.

    该公司最近一直在为美国的业务发展作准备。

24 PHRASE 守旧的;一成不变的;固执的 If you say that someone is set in their ways, you are being critical of the fact that they have fixed habits and ideas which they will not easily change, even though they may be old-fashioned. 守旧的;一成不变的;固执的 [disapproval]

26 PHRASAL VERB 建起;设立;设置 If you set up a temporary structure, you place it or build it somewhere. 建起;设立;设置

  • They took to the streets, setting up roadblocks of burning tyres.

    他们走上街头，用点燃的轮胎设置路障。

  • 200 peace activists are planning to set up a peace camp at the border.

    200名和平人士正计划在边界建一处和平营地。

27 PHRASAL VERB 安装，装配，调试(设备或机器) If you set up a device or piece of machinery, you do the things that are necessary for it to be able to start working. 安装，装配，调试(设备或机器)

  • Setting up the camera can be tricky.

    调试相机可能会很费事。

  • I set up the computer so that they could work from home.

    我把电脑设置好，这样他们就可以在家办公了。

28 PHRASAL VERB 立业;开业;创业 If you set up somewhere or set yourself up somewhere, you establish yourself in a new business or new area. 立业;开业;创业

  • The mayor's scheme offers incentives to firms setting up in lower Manhattan.

    市长的方案激励了打算在曼哈顿下城开业的公司。

  • He worked as a dance instructor in London before setting himself up in Bucharest.

    在布加勒斯特开公司之前，他在伦敦当舞蹈教练。

  • Grandfather set them up in a liquor business.

    爷爷资助他们经营酒业。

29 PHRASAL VERB 安(家);开(店) If you set up home or set up shop, you buy a house or business of your own and start living or working there. 安(家);开(店)

  • They married, and set up home in Ramsgate.

    他们结婚了，在拉姆斯盖特安家定居。

  • ...20 businessmen hoping to set up shop in Japan.

    希望在日本开店的20名商人

30 PHRASAL VERB 引起;引发;产生 If something sets up something such as a process, it creates it or causes it to begin. 引起;引发;产生

  • The secondary current sets up a magnetic field inside the tube.

    二次电流在管子内部会产生磁场。

  • This can help you satisfy the craving without setting up problems later on.

    这可以帮你满足欲望，而且事后又不会惹出问题。

31 PHRASAL VERB 使更健康;使更有活力;使状态更好 If something sets you up for something, it puts you in a good condition or position to deal with it, for example by making you feel healthy and energetic. 使更健康;使更有活力;使状态更好

  • I have my cornflakes and smell the fresh air and the grass and it sets me up for the day.

    我吃了些脆玉米片，呼吸着充满青草气息的新鲜空气，这让我一天都充满活力。

  • The win sets us up perfectly for the match in Belgium.

    这场胜利为我们在比利时的比赛鼓足了劲儿。

32 PHRASAL VERB 诬陷;陷害;冤枉 If you are set up by someone, they make it seem that you have done something wrong when you have not. 诬陷;陷害;冤枉

  • He claimed he had been set up after drugs were discovered at his home.

    在他家里发现毒品后，他声称自己是被人陷害的。

  • Maybe Angelo tried to set us up.

    也许安杰洛想要陷害我们。
