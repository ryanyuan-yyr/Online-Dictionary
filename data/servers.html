server
sɜːʳvəʳ ˈsɜrvər
Word forms: plural servers
1 N-COUNT 服务器 In computing, a server is part of a computer network which does a particular task, for example storing or processing information, for all or part of the network. 服务器 [computing]
2 N-COUNT [oft adjective NOUN] (网球或羽毛球的)发球者 In tennis and badminton, the server is the player whose turn it is to hit the ball or shuttlecock to start play. (网球或羽毛球的)发球者

  • ...a brilliant server and volleyer.

    一个发球和截球技术高超的发球上网型网球手

3 N-COUNT [oft n NOUN] (叉、匙等)上菜用具 A server is something such as a fork or spoon that is used for serving food. (叉、匙等)上菜用具

  • ...salad servers.

    夹色拉的叉匙
