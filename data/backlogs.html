backlog
bæklɒg -lɔːg ˈbækˌlɔg
Word forms: plural backlogs
N-COUNT 积压的事情；未完成的工作 A backlog is a number of things which have not yet been done but which need to be done. 积压的事情；未完成的工作

  • There is a backlog of repairs and maintenance in schools.

    对各学校的修缮和维护工作积压了一堆。

SYN build-up, stock, excess, accumulation
