seal off
1 PHRASAL VERB 封闭;隔离 If one object or area is sealed off from another, there is a physical barrier between them, so that nothing can pass between them. 封闭;隔离 [Also VERB n PREPOSITION]

  • Windows are usually sealed off. [be VERB-ed PREP.]

    窗户通常是封闭的。

  • ...the anti-personnel door that sealed off the chamber. [VERB PREP. noun]

    将会议室隔离起来以防外人进入的门

SYN isolate, segregate, quarantine, board up
