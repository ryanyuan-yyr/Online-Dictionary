set out
1 PHRASAL VERB 动身;出发;启程 When you set out, you start a journey. 动身;出发;启程 [Also VERB PREPOSITION]

  • When setting out on a long walk, always wear suitable boots. [VERB PREP. prep./adv.]

    每次出发远行，都要穿合适的鞋子。

SYN embark, set off, start out, begin  
2 PHRASAL VERB 开始;着手 If you set out to do something, you start trying to do it. 开始;着手

  • He has achieved what he set out to do three years ago. [VERB PREP. to-infinitive]

    他已经实现了他3年前的奋斗目标。

  • We set out to find the truth behind the mystery. [VERB PREP. to-infinitive]

    我们开始着手揭开谜团背后的真相。

SYN determine, decide, purpose, resolve  
3 PHRASAL VERB 安排;摆放;陈列 If you set things out, you arrange or display them somewhere. 安排;摆放;陈列 [Also VERB n PREPOSITION]

  • Set out the cakes attractively, using lacy doilies. [VERB PREP. noun]

    铺上网眼垫纸，将蛋糕摆得诱人些。

SYN arrange, present, display, lay out  
4 PHRASAL VERB (清晰而有条理地)陈述，阐述，说明 If you set out a number of facts, beliefs, or arguments, you explain them in writing or speech in a clear, organized way. (清晰而有条理地)陈述，阐述，说明 [Also VERB n PREPOSITION]

  • He has written a letter to The Times setting out his views. [VERB PREP. noun]

    他已经给《泰晤士报》去信阐明自己的观点。

  • You will be given a Back To Work plan which sets out how you can best help yourself. [VERB PREP. wh]

    你会收到一份《回归工作》计划，上面条理清晰地告诉你该如何将自己调整到最佳状态重返工作。

SYN explain, list, describe, detail
